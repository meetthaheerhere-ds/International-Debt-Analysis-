🌍 International Debt Analysis System

📌 Project Overview

This project analyzes global debt data using Python, MySQL, and Streamlit. It transforms raw World Bank data into meaningful insights through data cleaning, SQL analysis, and visualization.

---

🚀 Tech Stack

* Python (Pandas)
* MySQL
* Streamlit
* Plotly

---

📊 Features

* Data Cleaning & Preprocessing
* SQL-based Analysis (30 Queries)
* Interactive Dashboard
* Country-wise & Indicator-wise Insights
* End-to-End Data Pipeline

---

📁 Project Structure

* app.py → Data processing & MySQL insertion
* db.py → Streamlit dashboard
* mini project 2.sql → Database schema
* data/ → Dataset

---

▶️ How to Run

1. Install dependencies

```bash
pip install pandas mysql-connector-python streamlit plotly
```

2. Run data pipeline

```bash
python app.py
```

3. Run dashboard

```bash
streamlit run dashboard.py
```

📊 Key Insights

* Few countries contribute major portion of global debt
* External debt dominates overall financial indicators
* Debt trend shows variation across years
* Significant gap between high and low debt countries
